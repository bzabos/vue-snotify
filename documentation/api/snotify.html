remove

  • type: Function

    Signature:

    (
  id?: number
) => void

    If id passed, removes toast instantly. If no param passed it's the same as clear() If you want to remove toast with exit animation use vm.$snotify.$emit('remove', id)

clear

  • type: Function

    Signature:

    () => void

    Clear notifications array instantly
